What is being forecasted?

  • We are forecasting load, a measure of power, or energy per unit time, which illustrates the current demand for energy at that instant.

How does this work?

  • The user (you) inputs data into the text fields.
  • We run the data through the neural network, which gives us a load prediction.
  • Due to this fact, only have a few data inputs, so the forecast will be less accurate.

What does the plot show?

  • The shap plot shows the contributions of each variable to the final value.
  • Variables in red caused an increase in the expected price.
  • Variables in blue caused a decrease in the expected price.
