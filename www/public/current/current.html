Getting Price...

Getting Explainer Image...

What is being forecasted?

  • We are forecasting price, a measure of cost per unit energy.

How does this work?

  • We make an API call to OpenWeather and Entose to get current generation and weather data.
  • We run the data through the XGBoost model, to get a short range price.
  • Due to this fact, we have access to far more data, and can utilize a short term decision tree approach.

What does the plot show?

  • The shap plot shows the contributions of each variable to the final value.
  • Variables in red caused an increase in the expected price.
  • Variables in blue caused a decrease in the expected price.
